.sql
에 작성하는 sql문은 반드시 맨 뒤에 ;으로 sql 이 끝나는 마침표 와 같은
기호를 작성해야 한다.

.xml 에 작성한 sql 문은 ;을 작성하면 안된다.
mybatis 는 sql 내부에서 sql 작성을 끝낼 때 <> 태그로 이미 사용하겠다
표기하였기 때문에 따로 ;을 붙이지 않는다.


    <select id="모두조회">
        SELECT id, title, content FROM posts
    </select>

    정상적으로 작동

    <select id="모두조회">
        SELECT id, title, content FROM posts;
    </select>
    이미 <> 형태로 어디서 sql 이 시작하고 어디서 sql 문이 종료된다
    표기하였기 때문에 ; 을 붙이지 않는다.
